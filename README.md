# FlexMenFlutter
Реплика приложения The Flex на Dart/Flutter.
Это нерабочее приложение, оно лишь визуальный пример для мужского фитнес клуба. 
Но малый функционал приветствуется.

Тестировал на эмуляторе Android и на Web, на IOS не смог.
ThemeData ( темная, светлая ), Firebase ( Авторизация для идентификации пользователей, база данных с занятиями и занятиями у пользователей ), ListView.builder и FutureBuilder.
Без обработок исключений.
